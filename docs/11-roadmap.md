# MVP Roadmap

Roadmap peluncuran disusun berdasarkan prioritas penyampaian value bagi pengguna pilot (Angkringan).

### Phase 1: Core Foundation & POS (MVP Pilot Angkringan)
- Set up infrastruktur, CI/CD, Nuxt & Laravel Skeleton.
- Modul IAM (Tenancy, Roles, Auth).
- Modul Catalog & Base Inventory (CRUD Produk).
- POS Checkout Flow (Pembuatan Order).
- **Special Feature**: Alur Tutup Kasir & Rekomendasi Produksi.

### Phase 2: Operations & Finance
- Modul pencatatan Arus Kas (Pemasukan & Pengeluaran eksternal).
- Modul Purchasing (Pembuatan order barang ke supplier).
- Laporan Laba Rugi sederhana.

### Phase 3: CRM & Ekspansi Branch
- Manajemen multi-branch tingkat lanjut (Fitur Mutasi Stok antar Cabang).
- CRM: Basis data pelanggan (Customer History).
- Sistem Poin Loyalitas & Diskon Member.

### Phase 4: Enterprise Scale (Post-Pilot)
- Integrasi Payment Gateway (QRIS dinamis Midtrans).
- Notifikasi WhatsApp otomatis ke Owner (Shift Closing Report).
- Modul analitik tingkat lanjut.
